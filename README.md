## Features

- Join existing chat rooms or create your own.
- Exchange messages with other participants in real-time.
- Simple and intuitive user interface.
- Leave the chat room and disconnect when you're done.

## Technologies Used

- **Flask**: A micro web framework for Python.
- **Socket.IO**: Enables real-time, bidirectional communication between clients and the server.
- **HTML/CSS**: Used for creating the user interface.
- **JavaScript**: Handles client-side interactions and Socket.IO integration.

## Usage

1. Clone the repository:

        git clone https://github.com/Diptangshu4/chat_room_flask.git
        

2. Change into the project directory:

        cd chat_room_flask
        

3. Install the required dependencies:

        pip install -r requirements.txt
        
        
4. Run the application:

        python app.py
        

5. Open your web browser and visit [http://localhost:5000](http://localhost:5000) to access the chat room.
